Title

Crowdsourcing Documentary Transcription: an Open Source Tool

Description

Proposal to create an open source transcription tool to enable researchers to access handwritten historical documents. To be used with Papers of the War Department, 1784-1800 and a few other test sites.
(Budget Justification documents have been redacted)

Deliverables

1. To build an open source transcription tool that will allow users to contribute work to digital documentary editions.
2. To build a CMS adapter interface that allows that tool to function with standards compliant system.
3. To implement the tool with the Papers of the War Department project.
4. To test the tool with three project partners.
5. To author a Guide on Crowdsourcing Documentary Transcription that will provide both technical and practical advise to editors.
6. To disseminate the open source tool widely to digital archivists.
