Постановление № 155-17с Совета Министров Туркменской ССР
5 мая 1956 года
г. Ашхабад
 
Вопросы Уполномоченного Совета по делам религиозных культов при Совете Министров ТССР.
Совет Министров Туркменской ССР постановляет:
1. Одобрить мероприятия, представленные Уполномоченным Совета по делам религиозных культов при Совете Министров ТССР, по встрече зарубежных мусульман.
2. Поручить Уполномоченному Совета по делам религиозных культов при Совете Министров Туркменской ССР:
а) провести в июне 1956 года совещание (инструктаж) областных уполномоченных Совета по делам религиозных культов при облисполкомах по вопросам подготовки мечетей и духовных лиц на случай приезда мусульман из-за границы;
б) провести необходимую подготовительную работу по показу зарубежным мусульманам промышленных предприятий, музеев, культурно-просветительных учреждений, учебных заведений, медицинских учреждений и др.
3. Обязать министерства, ведомства, дирекции учебных заведений и облисполкомы по требованию Уполномоченного Совета по делам религиозных культов при Совете Министров ТССР провести в подведомственных предприятиях и учреждениях необходимые подготовительные работы для показа их представителям зарубежных мусульман.
4. Обязать исполкомы областных Советов депутатов трудящихся:
а) обеспечить приведение в надлежащее санитарное состояние территорий колхозов, МТС, жилища и дворы

колхозников по заявкам областных уполномоченных Совета по делам религиозных культов;
б) оказывать уполномоченным Совета по делам религиозных культов при облисполкомах необходимое содействие по организации встречи представителей зарубежных мусульман.
Зам. Председателя Совета Министров Туркменской ССР Г.Хорев
Управляющий Делами Совета Министров Туркменской ССР А.Оразов

Секретно
Приложение к Постановлению Совета Министров ТССР от 5 мая 1956 г. № 155-17с
Мероприятия
по встрече мусульман из-за границы
1. Договориться с Средне-Азиатским духовным управлением:
а) о проведении ремонта действующих мечетей, об озеленении и благоустройстве их территорий;
б) о приобретении необходимого количества молитвенного инвентаря (намазлик, кундуки и др.) для совершения религиозных обрядов, а также необходимой посуды и продуктов питания для устройства приемов представителей зарубежных мусульман;
в) для встречи и сопровождения представителей зарубежных мусульман подготовить из числа духовенства действующих мечетей соответствующих кандидатов, а также необходимое число экскурсоводов.
[2.] Наметить следующий примерный маршрут посещения зарубежными мусульманами промышленных предприятий, колхозов, совхозов, учреждений и учебных заведений:
а) музей изобразительных искусств;
б) Туркменский медицинский институт;
в) художественная ковровая мастерская;
г) Ашхабадский механизированный стеклозавод им. Профинтерна;
д) колхоз имени Молотова с/совета Багир;
е) механизированный стеклозавод им. Калинина.
Уполномоченный Совета по делам религиозных культов при Совете Министров ТССР А.Комеков
