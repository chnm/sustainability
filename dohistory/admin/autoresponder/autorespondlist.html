IUINC - Auto Responder Utility

This will allow you to create email auto responder. The auto responder will create a email alias that will receive your mail. This user must not exist in the Virtual Mail Manager Utility located at users2.iuinc.com.
Auto Responder Address:
For example, if you would info@yourdomain.com to give an auto response, simple type info in the above space. Remember, info@yourdomain.com must not exist in the Virtual Mail Manager Utility located at users2.iuinc.com.

Remember to include @yourdomain.com in the address.


Real Address:
This is a real address that you use that the sender requests will be forwarded to. Please include the full email address, include @yourdomain.com.

Subject of the response:
This subject will be shown in the response received by the sender. For example, "I'm on vacation!".

Reply Address:
This can be an alternate reply address shown to the sender. For example, if you are no longer using the address that the auto responder is pointed to, you may set a reply address so when the sender replies to the auto response, they will be emailing your new email address.

Blank Carbon Copy:
This can be a real address that you would like carbon copies of the auto response sent to.

Auto Response Message:

Type in your message for people to see when they receive the autoresponse.
