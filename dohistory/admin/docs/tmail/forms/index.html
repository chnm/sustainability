TMail Plug-N-Play Forms

The following is a list of pre-written TMail forms and templates. All scripts are ready to use AS IS. The only modifcation you need make is to the _MAILTO and _MAILCC variable; Simply set them to the e-mail addresses where you want the form sent. Remove the _MAILCC input field if there is to be noone CC'd with the form results.
  • Basic Request Information: Requires standard contact information.

  • Basic Request Information (WITHOUT TEMPLATE): If you want to add fields to the basic request and don't want to worry about updating the template, use this non-templated request information form. Any input variables added to the form will automatically be added into the form mailing.

  • Web Based E-Mail: On-line web page for e-mail like submissions.
