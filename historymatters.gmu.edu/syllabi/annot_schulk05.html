In an effort to make new media resources and the development of basic research and computer skills an integral part of my survey course from the very beginning, I have created two introductory exercises, an "Introduction to Search Engines" and an "Introduction to Web Site Evaluation." Step by step instructions for each of these exercises is handed out to the students in print form, with printouts of the successive computer screens they should see as they progress through each exercise. (Click here for an electronic version of the instructions for the "Introduction to Search Engines.") The "Introduction to Web Site Evaluation" requires that students begin by accessing a list of "Recommended American Civilization Web Sites for Students and Faculty" (www.digitalhistory.com/schools/PembrokeHillSchool/sfproj/amciv.html) After exploring several of the sites utilizing the active links that are provided, each student is required to choose a preferred Web site for in depth study. (Click here for an electronic version of the instructions for the "Introduction to Web Site Evaluation"). The evaluation criteria are provided by three helpful on-line resources:
a) "Evaluating information found on the Internet"
(http://milton.mse.jhu.edu:8001/research/education/net.html);
b) "Thinking Critically about World Wide Web Resources"
(http://www.library.ucla.edu/libraries/college/instruct/critical.html) and
c) "WWW CyberGuide Ratings for Web Site Design"
(http://www.cyberbee.com/guide2.html).
